# 更新紀錄

## 2026-06-17 - 音訊與載入修正版

狀態：準備同步到 GitHub 與 Vercel。

主要內容：
- 將指板按鍵事件改為單一 `pointer` 流程，避免手機同時觸發 touch 與 mouse 造成重複發音。
- 修正手機點音時可能出現的雜音與錄音回放雙重觸發問題。
- 正式網站恢復腳本快取，避免每次開啟都重新抓取所有腳本。
- 將頁面 favicon 與 apple touch icon 改成較小的 logo 檔案，降低初始載入負擔。

## 2026-06-17 - 穩定測試版

狀態：已同步到 GitHub，並更新到 Vercel 正式測試網址。

公開網址：
https://erhupitch-app.vercel.app/

主要內容：
- 完成二胡指板六個調性：D、G、C、Bb、F、A。
- 依照參考圖片調整指板格線、空弦、內弦與外弦標示。
- 調整手機版版面，保留上方店家資訊。
- 調整指板大小與置中，手機可上下滑動查看。
- 修正簡譜高音點、低音點的顯示位置。
- 加入二胡音色按鍵播放。
- 加入旋律錄音、單次播放、循環播放與清除錄音功能。
- 將調性資料、音訊、錄音、UI 元件拆分到 `src` 資料夾，方便後續維護。
- 更新手機 APP 圖示與 manifest。

注意事項：
- 若手機看到舊版，請重新整理，或在網址後面加上 `?v=1` 測試快取。
- 修改音位資料時，優先檢查 `src/app-data.js`。
- 修改指板樣式時，優先檢查 `index.html` 的 CSS。
- 修改聲音時，優先檢查 `src/audio.js`。

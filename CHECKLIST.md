# 上線前檢查表

每次要更新到 Vercel 正式網站前，請先照這份清單檢查。

## 1. 基本開啟

- [ ] 本機網址可以開啟：`http://localhost:4173/`
- [ ] 正式網址可以開啟：`https://erhupitch-app.vercel.app/`
- [ ] 手機可以開啟正式網址
- [ ] 手機沒有看到舊快取畫面

## 2. 調性與指板

- [ ] D 調可以正常顯示
- [ ] G 調可以正常顯示
- [ ] C 調可以正常顯示
- [ ] Bb 調可以正常顯示
- [ ] F 調可以正常顯示
- [ ] A 調可以正常顯示
- [ ] 內弦、外弦標示正確
- [ ] 空弦位置正確
- [ ] 格線間距固定，沒有忽大忽小
- [ ] 沒有音階資料的位置不顯示多餘音符

## 3. 簡譜與音名

- [ ] 簡譜數字符合參考圖片
- [ ] 低音點在數字正下方
- [ ] 低音點沒有遮到下方音名
- [ ] 一個高音點與數字距離適中
- [ ] 兩個高音點垂直排列，沒有和數字重疊
- [ ] 音名顯示清楚，沒有被點或圓圈遮住

## 4. 手機版畫面

- [ ] 店家資訊有顯示，沒有被隱藏
- [ ] 店家資訊排版不凌亂
- [ ] 工具列乾淨，調性選擇容易操作
- [ ] 指板置中
- [ ] 指板大小適合手機查看
- [ ] 可以上下滑動查看完整指板

## 5. 聲音與錄音

- [ ] 點擊音符可以發聲
- [ ] 聲音開頭沒有突兀金屬音
- [ ] 開始錄製可以正常記錄旋律
- [ ] 停止錄製後音符數量正確
- [ ] 播放旋律可以單次播放
- [ ] 循環播放可以正常開始與停止
- [ ] 清除錄音可以清空目前旋律
